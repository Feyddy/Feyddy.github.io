它可以加速网络的访问。但是却不能及时更新数据的显示。

POST方法和GET方法的区别

get:把数据放在URL（网址）里面来提交
post:把数据放在http content里面提交，在url里面不会显示

get：安全性很低，容易在历史记录里面看到自己的用户名等等、容量很低、便于分享
post：安全性一般、容量几乎无限
