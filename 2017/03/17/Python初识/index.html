元组(Tuple)

  • 创建只有一个元素的Tuple，需要用逗号结尾消除歧义；
  • Tuple是不可变的list，一旦创建了一个tuple就不能以任何方式改变他；
  • Tuple与list的相同之处：
    • 整个元素集都是用小括号包含的；
    • 都是按照定义的次序排序，索引也一样；
    • 负数的索引都是从尾部开始的；
    • 都可以用slice分片。注意当分割一个list时，会得到新的list，tuple也一样。
  • Tuple不存在的方法：
    • 不能增减元素；
      可以用in来查看一个元素是否存在与tuple中
  • 用Tuple的好处：
    • 比list操作快。遍历的时候尽量用Tuple；
    • 因为不能修改，所以安全；
    • 可以和list转换
    • 可以一次赋多个值

错误与异常处理(Error & Exceptions)

Python 有两种错误类型：

  • 语法错误(Syntax Error)
  • 异常(Error)

首先，try语句下的（try和except之间的代码）被执行；
如果没有出现异常，except语句将被忽略；
如果try语句之间出现了异常，try之下异常之后的代码将被忽略，直接跳跃到except语句；
如果异常出现，但并不属于except中定义的异常类型，程序将被执行外围一片的try语句；
如果异常没有被处理，将产生unhandled exception的错误。
