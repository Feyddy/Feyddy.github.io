字典：当索引不好用时

通过名字引用值的数据结构，成为映射。字典是Python中唯一內建的映射类型。字典没有顺序，但是都存储在一个特定的键里面。键可以使数字、字符串甚至是元组。

字典的使用

什么时候选择使用字典呢？

  • 表征游戏棋盘的状态，每个键都是由坐标值组成的元组；
  • 存储文件修改次数，用文件名作为键；
  • 数字电话/地址薄。

创建和使用字典

先来创建一个简单的字典：

字典是由多个键及与其对应的值构成的对组成（键值对）。名字是键，号码是值。每个键和值之间用:隔开，项之间用逗号隔开。整个字典用大括号括起来。空字典：{}。

基本字典操作

在很多方面和序列类似：

  • len(d)返回d中项（键-值对）的数量；
  • d[k]返回关联到键k上的值；
  • d[k]=v将值v关联到键k上；
  • del d[k]删除键为k的项；
  • k in d检查d中是否含有键为k的项；

字典和列表的一些重要区别：

  • 键类型：字典的键不一定是整型数据，也可能是其他不可变类型，比如浮点型、字符串型或者元组；
  • 自动添加：即使那个键起初在字典中并不存在，也可以为他分配一个值，这样字典就会建立新的项。而（在不使用append方法或者其他类似操作的情况下）不能讲值关联到列表范围之外的索引上。
  • 成员资格：表达式k in d(d为字典)查找的是键，而不是值。
